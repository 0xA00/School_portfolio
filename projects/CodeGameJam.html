Code Game Jam 2023

Description

This project was developed during the Code Gam Jam 2023 at the University of Montpellier. The theme was "Maitre du temps" and we had to create a game "from scratch" in under 30 hours! It was pretty challenging and our team managed to win the public prize and the jury's honorable mention! Perfoming under pressure was a new experience that felt fun at first, it was even greater to see our team achieving each goal we set through the night!

Technologies

  • Unity
  • C#
  • USB Drive
