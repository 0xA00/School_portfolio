Minecraft plugin - "RealSyncDate"

Description

This plugin was a project I thought about one day, it is really simple, this plugin let you synchronize your Minecraft daylight cycle to the one you live on. For example: if it's noon in real life it will also be noon in Minecraft. The plugin has an interface where you can choose the timezone you want to use, so you are not limited by the one that the server you play on uses. This project was pretty fun to do and it was challenging
