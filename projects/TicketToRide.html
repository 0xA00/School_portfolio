Ticket to ride Europe

Description

This game have been made during the 2nd semester of my 1st year, we had to recreate the board game "Ticket to ride - Europe" with Java using JavaFX as the graphic engine. Our Human Machine Interface classes had help us make this game because JavaFX was new to us and we learned a lot of things with thoses classes. On this project, my teammate and I have worked really hard to get the result we wanted.
